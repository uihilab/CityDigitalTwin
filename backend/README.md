# Backend Template

## Intro
Provides a NodeJS + Express backend template. Also provides a Python ecosystem within.

### Generic Actions

Install dependencies

```
npm install
```

Running in development

```
npm run dev
```

Running in production

```
npm start
```

Running tests / linting

```
npm test

npm run lint
```
